# fixed_deposit_calculator# FD Interest Calculator

## Overview

The FD Interest Calculator is a simple web application that helps users calculate the interest earned on a Fixed Deposit (FD) based on the deposit amount, interest rate, and time period. The project provides an interactive interface using range sliders for easy input selection.

## Features

* Interactive sliders for selecting:

  * Deposit Amount
  * Interest Rate
  * Time Period
* Real-time display of selected values
* One-click interest calculation
* Simple and responsive user interface
* Built using HTML, Bootstrap, and JavaScript

## Technologies Used

* HTML5
* CSS3 (Bootstrap 5)
* JavaScript

## How It Works

1. Select the Deposit Amount using the slider.

2. Select the Interest Rate (%).

3. Select the Time Period (Years).

4. Click the **Calculate** button.

5. The application calculates the simple interest using the formula:

   Interest = (Principal × Rate × Time) / 100

6. The calculated interest is displayed in the result field.

## Formula Used

Simple Interest:

SI = (P × R × T) / 100

Where:

* P = Principal Amount
* R = Annual Interest Rate
* T = Time Period (Years)
